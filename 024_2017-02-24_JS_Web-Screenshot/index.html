Take a screenshot of an element on this page! With DomOutline and html2canvas

This script lets you take screenshot of elements on this page with one click. Just click the button in the right hand corner and then select an element. After clicking on it a png screenshot will open in a new tab.
If the <img> is from another origin, this script needs a html2canvas proxy. Beside this, the script uses jQuery.DomOutline to select the element.
